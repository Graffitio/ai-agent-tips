👋 오늘의 한마디

복잡한 작업을 AI에게 맡길 때, 한 번에 모든 걸 요청하면 결과는 얕아집니다. "프롬프트를 설계하는 방식"과 "작업을 단계별로 연결하는 방식"이 결과를 완전히 바꿉니다. 오늘은 이 두 가지를 마스터하는 날입니다.

🎯 오늘의 팁 2선
#057 📚 중급 프롬프트 설계

CRISPE 프레임워크로 프롬프트를 구조화하기

CRISPE는 Capacity, Role, Insight, Statement, Personality, Experiment의 앞글자를 모은 프롬프트 설계 공식입니다. 이 6가지 요소를 순서대로 포함하면, AI가 당신의 의도를 정확히 이해하고 맥락에 맞는 답변을 생성합니다.

각 요소의 역할:
• Capacity: AI가 어떤 능력을 발휘해야 하는가 (분석가, 마케터, 코더 등)
• Role: AI가 어떤 입장에서 답변하는가 (전문가, 초보자, 스타트업 컨설턴트 등)
• Insight: 문제 상황의 배경과 맥락
• Statement: 명확한 요청과 원하는 결과물의 형태
• Personality: 톤, 스타일, 표현 방식 (전문적, 친근한, 간결한 등)
• Experiment: 결과를 검증하고 개선하는 방식

💬 CRISPE 적용 예시
✅ CRISPE 구조로 작성한 프롬프트: Capacity: 당신은 D2C 마케팅 전문가입니다. Role: 성공적인 반려동물 용품 브랜드를 여러 개 론칭한 경험이 있는 전략가 입장에서 답변하세요. Insight: 우리는 월 광고 예산 500만원으로 시작하는 반려견 간식 신규 브랜드입니다. 경쟁사는 10곳 이상이고, 우리의 차별점은 '자연식재료 + 합리적 가격'입니다. Statement: 3개월간의 인스타그램 & 유튜브 마케팅 로드맵을 월별 액션 플랜 형태로 구성해주세요. 각 월별로 예산 배분, 콘텐츠 유형, KPI까지 포함하세요. Personality: 실무적이고 즉시 실행 가능한 톤으로, 과도한 설명 없이 핵심만 전달하세요. Experiment: 첫 답변 후 피드백을 반영할 수 있도록, 각 월별 전략의 근거를 3줄 이내로 설명해주세요.

#058 📚 중급 작업 파이프라인

체인 프롬프트로 복잡한 작업을 단계별로 분리하기

체인 프롬프트(Chain Prompting)는 하나의 큰 작업을 여러 개의 작은 프롬프트로 나누어, 이전 단계의 결과를 다음 단계의 입력으로 사용하는 기법입니다. 각 단계에서 AI의 답변을 검토하고 확인할 수 있어서, 최종 결과물의 질이 비약적으로 높아집니다.

체인 프롬프트의 3가지 원칙:
1. 단계 분명화: "1단계, 2단계..." 명시적으로 표현
2. 결과 검증: 각 단계가 끝난 후 피드백 반영
3. 문맥 이월: 이전 단계의 결과를 다음 프롬프트에 포함

💬 체인 프롬프트 실행 순서
📋 복잡한 작업: 신제품 출시 마케팅 전략 수립 ❌ 잘못된 방법 (한 번에 다 물어보기): "신제품 출시 전략을 마세요. 경쟁 분석, 타겟 정의, 메시징, 채널 선택, 예산 배분까지 모두 포함해줘." → 모든 내용이 피상적임 ✅ 올바른 방법 (체인 프롬프트로 단계화): [1단계] 경쟁사 분석 "우리 제품의 주요 경쟁사 3곳을 정의하고, 각각의 강점·약점·포지셔닝을 정리해줘." → 결과 검토 후 피드백 [2단계] 타겟 정의 (1단계 결과 활용) "위 경쟁 분석을 바탕으로, 우리가 공략할 타겟 고객을 3가지 페르소나로 정의해줘. 각 페르소나별 구매 동기도 포함." → 결과 검토 후 피드백 [3단계] 메시징 개발 (1,2단계 결과 활용) "이 타겟들에게 우리 제품의 차별점을 어떤 메시지로 전달할지, 3가지 버전의 핵심 메시징을 만들어줘." → 결과 검토 후 피드백 [4단계] 채널&예산 (1,2,3단계 결과 활용) "각 타겟별로 가장 효과적인 마케팅 채널을 추천하고, 200만원 예산을 어떻게 배분할지 정리해줘."

💡 오늘의 인사이트

CRISPE와 체인 프롬프트는 서로 다른 차원의 도구입니다. CRISPE는 "한 번의 프롬프트를 어떻게 작성할 것인가"를 다루고, 체인 프롬프트는 "여러 프롬프트를 어떻게 연결할 것인가"를 다룹니다. 둘을 함께 사용하면, AI는 마치 당신의 생각을 읽는 것처럼 정확하고 깊이 있는 결과를 만들어냅니다. 복잡한 프로젝트일수록, 이 두 기법의 위력이 두드러집니다.

📖 오늘의 용어
CRISPE
CRISPE Framework
프롬프트를 구조화하기 위한 6가지 요소(Capacity, Role, Insight, Statement, Personality, Experiment)를 담은 프레임워크. AI의 응답 품질을 크게 높이는 설계 방법론입니다.
체인 프롬프트
Chain Prompting
복잡한 작업을 여러 개의 단계로 분리하고, 각 단계의 결과를 다음 프롬프트의 입력으로 활용하는 기법. 단계별 검증이 가능해 최종 결과의 정확도가 향상됩니다.
