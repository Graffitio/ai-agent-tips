🎯 오늘의 팁 2선
#023 📚 중급 템플릿 설계

프롬프트 템플릿 만들기

같은 형식으로 반복되는 작업이 있나요? 프롬프트를 템플릿화하면 변수만 바꿔가며 쓸 수 있습니다. 예를 들어 매일 다른 상품에 대한 소셜 미디어 카피를 작성한다면, 카피 작성 프롬프트를 한 번 완성한 뒤 상품명, 특징, 타겟만 바꿔 재사용하면 됩니다.

핵심은 [변수] 자리를 명확히 표시하는 것입니다. 프롬프트의 구조는 고정하되, 상황별로 달라지는 정보만 변수로 처리하세요. 이렇게 하면 일관되면서도 빠른 결과물을 얻을 수 있습니다.

💬 템플릿화 예시
[소셜 미디어 카피 작성 템플릿] 너는 D2C 브랜드의 마케팅 카피라이터야. [상품]: [상품명] [주요 특징]: [특징 3가지] [타겟 고객]: [타겟 설명] [톤]: [원하는 톤] 위 정보를 바탕으로: 1. 인스타그램 릴스 15초 스크립트 2. 틱톡 30초 영상 자막 3. 블로그 포스트 제목(3개) 를 작성해줘. --- 사용 예시: [상품]: 강아지 관절 영양제 [주요 특징]: 초록입홍합, MSM, 비타민D [타겟 고객]: 50대 반려견 보호자, 성분 중심 [톤]: 따뜻하고 신뢰감 있는

#024 📚 중급 조건 분기

조건문 넣기: if-then 프롬프트

"만약 상황이 A라면, B를 하고; C라면, D를 해줘" 이렇게 조건문을 넣으면 AI는 상황별로 다른 대응을 합니다. 한 번의 프롬프트로 여러 경우의 수를 처리할 수 있어 매우 효율적입니다.

고객 문의 대응, 콘텐츠 톤 결정, 우선순위 판단 등 상황에 따라 다른 처리가 필요한 모든 작업에 활용할 수 있습니다. 핵심은 조건을 명확하게, 각 경우의 대응을 구체적으로 작성하는 것입니다.

💬 If-Then 프롬프트 예시
[고객 문의 대응 자동화] 아래 고객 문의를 읽고, 다음 규칙에 따라 응답 템플릿을 작성해줘. [규칙] - 만약 문의 내용이 "제품 반품/환불"이라면: → 환불 정책(30일 이내 100% 환불)을 명시하고 반품 절차(3단계)를 친절하게 설명 - 만약 문의 내용이 "배송 지연"이라면: → 미안을 표현하고, 현재 배송 상태를 확인해주고 보상안(다음 구매 10% 할인) 제시 - 만약 문의 내용이 "제품 성분/효능"이라면: → 해당 성분의 효능을 과학적으로 설명하고 복용 방법과 주의사항 명시 - 만약 그 외 일반 문의라면: → 감사 인사 + 답변 + 추가 도움 제안 [고객 문의] "배송이 벌써 일주일이 넘었는데 아직도 안 와요. 빨리 받고 싶은데..." → 위 규칙에 맞는 응답 템플릿 생성

💡 오늘의 인사이트

프롬프트 템플릿과 if-then 조건문은 AI를 "일회용 답변 도구"에서 "반복 업무 자동화 시스템"으로 변신시킵니다. 처음에는 프롬프트 작성에 시간이 걸리지만, 한 번 완성되면 같은 구조의 작업을 몇 초면 처리합니다. 특히 마케팅, 고객 응대, 콘텐츠 제작 등 반복성 높은 업무에서 엄청난 시간 절약 효과를 볼 수 있습니다. 내 업무 중 매주 반복되는 것이 뭔지 한 번 찾아보세요.

조건문 프롬프트
Conditional Prompt / If-Then Logic
상황에 따라 다른 대응을 하도록 AI에게 미리 규칙을 정해주는 프롬프트. "만약 A라면 B를 하고, C라면 D를 해" 형식으로 여러 경우의 수를 한 번에 처리합니다.
