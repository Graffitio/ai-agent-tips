👋 오늘의 한마디

노코드 도구가 발전하면서 이제 코드 없이도 복잡한 워크플로우를 자동화할 수 있습니다. 하지만 구글 워크스페이스의 진정한 힘은 Google Apps Script(GAS)에 있습니다. 오늘은 GAS와 AI를 연동해 문서·시트·메일 자동화를 극단적으로 간단하게 만드는 방법을 배웁니다.

🎯 오늘의 팁 2선
#153 📘 중급 GAS + AI

Google Apps Script + AI 연동으로 구글 워크스페이스 자동화

Google Apps Script는 구글 클라우드 위에서 실행되는 자동화 언어입니다. GAS + Claude/GPT API를 결합하면, 구글 시트의 데이터를 AI로 분석하고 자동으로 문서를 생성할 수 있습니다.

예시: 영업 이메일 목록 → AI가 각 고객에 맞게 자동 작성 → 구글 드라이브에 저장. 이 모든 것이 한 번의 버튼 클릭으로 실행됩니다.

💬 실제 활용 패턴
// 구글 시트에서 고객 데이터 읽기 + AI로 제안서 자동 생성 function generateProposalWithAI() { const sheet = SpreadsheetApp.getActiveSheet(); const data = sheet.getRange("A2:C100").getValues(); data.forEach(row => { const [name, company, budget] = row; // Claude API 호출 const prompt = `${company}의 ${name}님께 예산 ${budget}원 범위의 제안서를 작성해줘`; const proposal = callClaudeAPI(prompt); // 구글 닥스에 자동 저장 createGoogleDoc(name, proposal); }); } → 매주 자동 반복 실행 가능 (트리거 설정)

#154 🔥 고급 n8n

n8n으로 셀프호스팅 AI 워크플로 구축

n8n은 오픈소스 자동화 플랫폼으로, 자신의 서버에 설치해 완전히 자유로운 워크플로우를 만들 수 있습니다. Zapier처럼 비용 부담 없이, 더 깊은 커스터마이징이 가능합니다.

GAS의 한계를 벗어나고 싶다면, n8n + 오픈소스 LLM(Ollama 등)을 조합해 온프레미스 AI 자동화 환경을 만들 수 있습니다.

💬 실제 활용 패턴
n8n 워크플로우 예시: [Slack 메시지 받기] ↓ [메시지를 Ollama LLM으로 분석] ↓ [감정 분석 결과에 따라 다른 채널로 라우팅] ↓ [결과를 구글 시트에 기록 + 알림 발송] 장점: ✓ 서버 비용만 고려 (API 비용 없음) ✓ 데이터 프라이버시 (로컬 실행) ✓ 무제한 자동화 (Zapier는 월정액/task 제한) ✓ 완전 커스터마이징 가능

💡 오늘의 인사이트

GAS vs n8n — 어떤 것을 선택할까요? GAS는 빠르고 간단하지만 구글 에코시스템 내 활용이 중심입니다. n8n은 깊은 자유도를 원하는 조직에 이상적입니다. 대부분의 팀은 GAS부터 시작해 복잡한 워크플로는 n8n으로 확장하는 하이브리드 방식을 채택합니다. 둘 다 배워두면, 자동화할 수 없는 업무가 거의 없어집니다.

📖 오늘의 용어
Google Apps Script
GAS
구글 클라우드 위에서 실행되는 JavaScript 기반 자동화 언어. 구글 시트, 닥스, 메일, 캘린더 등을 프로그래밍으로 제어할 수 있습니다. 학습 곡선이 낮고 구글 서비스와의 통합이 매우 자연스럽습니다.
n8n
n8n Workflow Automation
오픈소스 워크플로우 자동화 플랫폼. 자신의 서버에 설치해 운영하며, 200개 이상의 앱·서비스와 연동 가능합니다. 노코드 인터페이스와 함께 JavaScript 커스텀 코드도 지원해 깊은 자동화가 가능합니다.
