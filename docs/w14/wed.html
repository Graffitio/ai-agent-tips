PHASE 3 · 에이전트 심화
👋 오늘의 한마디

에이전트는 강력하지만 위험합니다. 무한 루프에 빠지거나 잘못된 도구를 선택하는 실패 모드를 이해하고, 그에 대한 안전장치를 구축하는 것이 Phase 3의 핵심입니다. 동시에 Claude의 MCP 서버를 활용하면 에이전트를 현실의 도구들과 연결할 수 있습니다. 오늘은 이 두 가지를 함께 다룹니다.

🎯 오늘의 팁 2선
#135 ⚡ 고급 안전 설계

에이전트의 실패 모드와 안전장치

에이전트가 스스로 도구를 선택하고 행동하는 과정에서 두 가지 주요 실패가 발생합니다. 첫째, 무한 루프 — 같은 도구를 반복 호출하거나 이전 상태로 돌아가는 악순환. 둘째, 도구 오용 — 부적절한 도구를 선택하거나 잘못된 매개변수를 전달하는 것입니다.

이를 방지하기 위해 세 가지 안전장치를 구현하세요: 1) 최대 스텝 제한 (예: 10회 이상 반복 금지), 2) 도구 접근 제어 (특정 도구는 특정 조건 하에서만 호출), 3) 상태 추적 (이전 상태와 현재 상태를 비교해 루프 감지).

💻 코드 예시
class SafeAgent: def __init__(self, max_steps=10): self.max_steps = max_steps self.step_count = 0 self.previous_state = None def run(self, task): while self.step_count < self.max_steps: action = self.decide_action() # 루프 감지: 이전 상태와 동일 if self.state == self.previous_state: raise LoopDetectedError() self.previous_state = self.state self.step_count += 1 self.execute(action) raise MaxStepsExceededError()

#136 🌤️ 중급 MCP 기초

Claude에서 MCP 서버 활용하기

MCP(Model Context Protocol)는 Claude를 외부 서비스나 데이터베이스와 연결하는 표준 프로토콜입니다. API를 직접 호출하는 것보다 훨씬 간단하고, Claude가 필요한 시점에 자동으로 적절한 도구를 선택합니다.

가장 간단한 시작은 stdio 기반 MCP 서버를 만드는 것입니다. Claude가 서버에 명령을 보내고, 서버가 응답을 반환하는 방식입니다. 예를 들어 날씨 정보, 데이터베이스 쿼리, 이메일 발송 등을 MCP로 구현할 수 있습니다.

💻 MCP 서버 예시
#!/usr/bin/env python3 import json import sys from mcp.server import Server server = Server("weather-mcp") @server.call_tool("get_weather") def get_weather(city: str): # 외부 API 호출 또는 로컬 데이터 조회 return { "city": city, "temperature": 22, "condition": "sunny" } if __name__ == "__main__": server.run()

💡 오늘의 인사이트

Phase 3에 들어서면서 에이전트는 더 이상 제어된 환경의 장난감이 아닙니다. 실제 시스템과 연결되고, 실제 작업을 수행합니다. 때문에 안전장치는 선택이 아닌 필수입니다. 동시에 MCP를 통해 에이전트를 강력한 도구들과 연결하면, 단순한 챗봇을 넘어 자동화 워크플로우 전체를 구축할 수 있게 됩니다. 오늘 배운 두 가지 — 안전성과 연결성 — 은 향후 4주 동안의 모든 심화 주제의 토대가 될 것입니다.

📖 오늘의 용어
무한 루프 (Infinite Loop)
Infinite Loop
에이전트가 같은 상태에 머물면서 같은 도구를 반복 호출하거나, 이전 상태로 계속 되돌아가는 현상. 예를 들어 "정보 조회 → 실패 → 다시 정보 조회"가 끝없이 반복되는 것.
MCP (Model Context Protocol)
Model Context Protocol
Claude 같은 LLM을 외부 도구, API, 데이터베이스와 연결하기 위한 표준 프로토콜. 에이전트가 필요한 시점에 자동으로 외부 리소스에 접근할 수 있게 해줍니다.
