PHASE 2 · 실무 적용
👋 오늘의 한마디

채용의 첫 단계는 채용공고(JD)에서 시작됩니다. 명확하고 매력적인 JD는 좋은 인재를 끌어당기고, 일관된 스크리닝 기준은 공정한 평가를 보장합니다. 오늘은 HR·PM이 함께 알아야 할 JD 작성과 이력서 평가의 실무 노하우를 다룹니다.

🎯 오늘의 팁 2선
#121 🌱 입문 JD 작성

직무기술서(JD) 작성 도우미 — 매력적이고 정확한 JD 만들기

좋은 JD는 직무의 현실을 담아냅니다. 일반적인 설명이 아닌, 실제로 이 직원이 하루하루 맡을 일들이 무엇인지 구체적으로 써야 합니다. 지원자가 자신의 역량을 정확히 판단하고, 입사 후 갭을 줄일 수 있기 때문입니다.

핵심 항목: ①주요 책임 ②필수 요구사항 ③우대사항 ④성공의 기준 ⑤팀 문화. 특히 "성공의 기준"은 지원자가 가장 알고 싶은 부분입니다. "첫 3개월 동안 이것을 이루면 성공"이라는 명확한 목표를 제시하세요.

💬 JD 체크리스트
✅ 필수 포함 요소: 1. 직무 개요 (1-2문장) → "고객 데이터 분석 기반 마케팅 전략 수립" 2. 주요 책임 (5-7개, 구체적인 액션) → "월간 캠페인 성과 분석 리포트 작성" → "고객 세그먼트 분석 및 타겟팅 전략 제안" 3. 필수 요구사항 (경력, 기술, 성향) → "관련 경력 3년 이상" → "SQL 또는 Python 실무 경험" → "논리적 사고와 커뮤니케이션" 4. 성공 지표 (3개월 후) → "첫 캠페인 분석 리포트 완성" → "팀원들로부터 신뢰 구축" 5. 팀 문화 (3-4개 키워드) → "데이터 기반 의사결정", "주인의식"

#122 📊 중급 스크리닝

이력서 스크리닝 기준 설계 — 공정하고 일관된 평가 기준

이력서 스크리닝은 기준이 명확할수록 공정합니다. "감으로" 판단하면 지원자마다 다른 잣대를 적용하게 됩니다. 미리 "반드시 필요한 것" vs "있으면 좋은 것"을 정의하고, 각 항목마다 점수 기준을 세워야 합니다.

세 단계 스크리닝 모델: ①K.O 기준(필수) → ②점수 평가(1-5점) → ③최종 면접 대상자 선정. 이렇게 하면 주관성이 줄어들고, 모든 평가자가 일관되게 평가합니다.

💬 스크리닝 기준 템플릿
📋 마케팅 PM 채용 — 스크리닝 기준 【 K.O 기준 (하나라도 NO면 탈락) 】 □ 관련 경력 2년 이상? (필수) □ 데이터 분석 경험 있음? (필수) 【 평가 항목 (1-5점) 】 1. 경력의 관련성 (1점: 약간, 5점: 매우 높음) - 5점: 마케팅 PM 직무 3년+ - 3점: 마케팅 직무 2-3년, 유사 경험 - 1점: 초급 경력 또는 관련성 낮음 2. 성과 사례의 명확성 (1점: 모호함, 5점: 구체적) - 5점: "캠페인 ROI 35% 개선" (수치화된 성과) - 3점: "캠페인 개선 경험" (일반적) - 1점: "마케팅 일함" (구체성 없음) 3. 리더십/협업 사례 (1점: 없음, 5점: 우수) - 5점: 팀 리드 또는 크로스펑셔널 프로젝트 주도 - 3점: 팀 협업 경험 - 1점: 개인 업무 중심 【 합계 계산 】 K.O 통과 후, 평가항목 점수 합계 - 13점 이상: 1순위 (면접 진행) - 10-12점: 2순위 (필요시 면접) - 9점 이하: 불합격

💡 오늘의 인사이트

좋은 채용은 투명성에서 시작됩니다. 명확한 JD는 지원자에게 "우리는 이런 사람을 찾고 있다"는 신호를 보내고, 일관된 스크리닝 기준은 "우리는 공정하게 평가한다"는 신뢰를 만듭니다. 결과적으로 지원자의 질도 높아지고, 입사 후 이탈도 줄어듭니다. 이번 주 피드백: 팀 내에서 스크리닝 기준을 함께 논의해 보세요. 기준을 만드는 과정 자체가 팀의 "이상적인 팀원상"을 재정의하는 기회가 됩니다.

📖 오늘의 용어
직무기술서(JD)
Job Description
특정 직무의 목적, 책임, 필요한 역량을 문서화한 것. 채용공고의 핵심 바탕이 되며, 입사 후 성과평가의 기준도 됩니다.
K.O 기준
Knock-Out Criteria
하나라도 충족하지 못하면 즉시 탈락되는 기준. 스크리닝 단계에서 불필요한 검토를 줄이고 시간 효율성을 높입니다.
